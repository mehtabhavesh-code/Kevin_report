# Genre Taxonomy Report.
 
## How to use
1. Open HTML
2. Use Preloaded JSON / Import another JSON of the same format
3. Modify the hierarchy (add/modify/delete nodes)
4. Export the new taxonomy as JSON.

## Taxonomy Structure
- Taxonomy is a 3-level hierarchichal JSON with Genre, Sub-Genre & Sub-Sub-Genre
- Each node contains
  - Definiton
  - Key Themes
  - Examples
  - Comparison Keys
